  <h1>Commander Token Board Guide</h1>
  <p><strong>Commander Token Board</strong> is a lightweight, mobile-optimized web application designed to help Magic: The Gathering players manage complex board states without needing physical cards or dice.</p>

  <h2>üì± Installation (iOS & Android)</h2>
  <p>For the best experience, install this app to your home screen. This removes the browser address bar and runs the app in full-screen mode.</p>
  <ol>
    <li>Open the website in <strong>Safari</strong> (iOS) or <strong>Chrome</strong> (Android).</li>
    <li>Tap the <strong>Share</strong> button (iOS box with arrow) or the <strong>Menu</strong> dots (Android).</li>
    <li>Select <strong>"Add to Home Screen"</strong>.</li>
    <li>Launch the app from the new icon on your device.</li>
  </ol>

  <h2>‚öîÔ∏è Adding Tokens</h2>
  <p>There are three ways to populate your board:</p>

  <h3>1. Quick Add (Search)</h3>
  <ul>
    <li>Tap the input box in the sidebar that says <em>"Start typing to find tokens..."</em>.</li>
    <li>Type a name (e.g., "Goblin", "Treasure").</li>
    <li><strong>Tap the result</strong> (Left Click) to instantly spawn that token.</li>
    <li><strong>Double Tap (Right Click)</strong> a result to configure stats/quantity before adding.</li>
  </ul>

  <h3>2. Deck Import (Recommended)</h3>
  <p>This scans your decklist and automatically sets up your Quick Add shortcuts with the correct tokens and artwork.</p>
  <ol>
    <li>Open the <strong>Menu <span class="icon">‚ò∞</span></strong> and select <strong>+ Add Deck</strong>.</li>
    <li>Paste your decklist (from Moxfield, Archidekt, Arena, etc.).</li>
    <li>Tap <strong>Import</strong>.</li>
    <li>The app fetches data from Scryfall. The <strong>Top 4</strong> most frequent tokens will appear at the top of your list automatically.</li>
  </ol>

  <h3>3. Custom Creator</h3>
  <ul>
    <li>Tap the <strong>"Custom Token"</strong> header in the sidebar to expand the form.</li>
    <li>Manually enter Name, Color, Power, Toughness, and Quantity.</li>
    <li>Optionally upload an image or check "Haste".</li>
  </ul>

  <h2>üîÑ The Next Turn Button</h2>
  <p>When you have tokens on the board, a <strong>"Next Turn <span class="icon">‚Ü∫</span>"</strong> button appears in the bottom right corner.</p>
  <p>Clicking this will:</p>
  <ul>
    <li><strong>Untap</strong> all tokens.</li>
    <li><strong>Remove Summoning Sickness</strong> (The <span class="badge-zzz">ZZZ</span> badge disappears).</li>
    <li><strong>Consolidate Stacks:</strong> Merges identical tokens back together (e.g., 3 tapped Goblins + 2 untapped Goblins become 5 Untapped Goblins).</li>
  </ul>

  <h2>üé® Artwork & Images</h2>
  <p>The app uses a "Smart Priority" system to decide what image to show:</p>
  <ol>
    <li><strong>Manual Upload:</strong> If you uploaded an image, that always takes priority.</li>
    <li><strong>Local File:</strong> Looks for a file named <code>Name.png</code> in the app folder.</li>
    <li><strong>Scryfall (Automatic):</strong> If no local image is found, it uses the official art found during the <strong>Deck Import</strong> process.</li>
  </ol>

  <h2>üíæ Saving & Offline Use</h2>
  <ul>
    <li><strong>Auto-Save:</strong> The board state saves automatically. You can close the app and return later.</li>
    <li><strong>Reset:</strong> Open the <strong>Menu <span class="icon">‚ò∞</span></strong> and select <strong>Reset Board</strong> to clear everything.</li>
    <li><strong>Offline:</strong> The app works fully offline (Deck Import requires internet once).</li>
  </ul>
